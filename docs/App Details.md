# App Main Details

# App Overview
A route map that displays traffic 
App has three dashboards one for PUV/PUJ, one for Private Drivers and one for commuters

# App Features
The app has an account system which has three types: PUV/PUJ Driver, Private Driver and Commuter

# Dassboard Features

### PUV/PUJ Driver Dashboard
1. A map since they have a fixed route it will just be traffic details
2. (optional feature) if the puv/puj have a RF card payment feature they can opt in to integrate if they do they can track how many passengers are in based on the amount of seats their unit has.
