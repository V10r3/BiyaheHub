# App Main Details

# App Overview
A route map that displays traffic 
App has three dashboards one for PUV/PUJ, one for Private Drivers and one for commuters

# App Features
The app has an account system which has three types: PUV/PUJ Driver, Private Driver and Commuter

# Dassboard Features

### PUV/PUJ Driver Dashboard
1. A map since they have a fixed route it will just be traffic details
2. (optional feature) if the puv/puj have a RF card payment feature they can opt in to integrate if they do they can track how many passengers are in based on the amount of seats their unit has.

### Private Driver Dashboard
1. An interactive map where they can plot their destination and for their start location they can either type their destination, manually pick on the map or use their current gps location it also shows traffic.
2. When there is heavy traffic a notification shows and asks if they want to go for an alternate route.

### Commuter Dashboard
1. An interactive map where they can plot their destination and for their start location they can either type their destination, manually pick on the map or use their current gps location and it also shows traffic.
2. There is a clock that they can set for when they want to arrive at said location.
3. The map shows jeeps, buses, taxis and trains (if applicable).
4. For jeeps they can select jeepneys based on their designation e.g. 01K Urgello to Parkmall
5. For buses it tracks the buses and shows stops for pickup and drop off.
6. For their choosen ride if the bus/puv did integrate the optional feature it shows the number of occupied/unoccupied seats.
7. For taxis it tracks them and shows pick up and unload areas.
8. If the user picks PUJ/Bus they are shown the best route with the best amount of rides and fare.
8. And if there is traffic it notifies the user for an alternate route/ride.
